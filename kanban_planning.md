# 1. 目標摘要
本專案旨在建立一個基於 Django 的 **「Vibe Coding 中文看板系統」**。讓使用者能將模糊的想法快速轉化為結構化的中文開發看板任務，實現從靈感到 MVP 的無縫銜接。

# 2. MVP 範圍
- **核心資料模型**：建立看板 (Board)、欄位 (Column) 與卡片 (Card) 的 Django Models。
- **基礎視圖**：展示看板清單與單一看板詳細內容。
- **UI 界面**：使用 Tailwind CSS 打造具備 Trello 風格的看板佈局。
- **初步整合**：將您提供的中文看板系統提示詞作為系統核心邏輯，用於產出卡片內容。

# 3. 功能模組
- **資料儲存模組**：負責看板、任務卡片的 CRUD。
- **視圖呈現模組**：處理前端看板畫面的渲染與導向。
- **AI 整合模組**：利用 `vibe-coding-kanban.md` 提示詞，將使用者輸入轉換為資料庫物件。
