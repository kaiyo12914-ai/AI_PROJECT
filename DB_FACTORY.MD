# DB Factory Config

此檔案可放 DB 連線設定，`webapps/database/db_factory.py` 會優先讀取。  
用途：把 DB 參數集中管理，不必放在 `.env`。

## 支援鍵

- 全域鍵：
  - `ORA_*`
  - `SQL_SERVER_*`
  - `SYBASE_*`
- DOC 廠別 profile：
  - `DOC_DB_{PROFILE}_{KEY}`
  - `PROFILE`：`MPC`, `202`, `205`, `209`, `401`

## 優先序

1. `DB_FACTORY.MD` 內鍵值  
2. `.env` 內同名鍵值  
3. 程式預設值

## ENV 規範（避免誤走 MOCK）

- `ENV=INT`：**強制使用實體 DB**（即使有 `FORCE_MOCK_DB=1` 也不走 mock）
- `ENV=EXT`：使用 mock JSON（停用外部 DB）
- `ENV=DEV/LOCAL/TEST/CI`：可用 `FORCE_MOCK_DB=1` 手動切 mock

建議內網正式環境設定：

```env
ENV=INT
FORCE_MOCK_DB=0
```
