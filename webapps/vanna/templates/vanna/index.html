Vanna 2.0 NL2SQL 智慧查詢工作區

PostgreSQL / Oracle 唯讀安全查詢端點
歡迎來到 **NL2SQL 查詢工作區**！👋
您可以在下方輸入自然語言問題，AI 會自動比對向量資料庫中的 Schema 與範例，為您生成對應的 SQL 查詢。
所有生成的 SQL 皆會經過 **SQL Guard (AST 解析)** 的安全審查，唯讀且僅限 `SELECT` 操作，以確保資料庫安全。
