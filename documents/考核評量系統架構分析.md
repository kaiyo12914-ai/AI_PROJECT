# 資訊工程部績效評語自動生成系統 - 系統架構文檔

## 1. 簡介
該模組為一套**半自動化績效評語生成功能**，針對企業員工績效考核提供軍事風格標準化評論。透過大語言模型（LLM）根據用戶選擇的績效等級和特質，快速生成符合規範的文字評語。

---

## 2. 整體架構

### 2.1 前端組件
#### 主要頁面：`templates/comment/performance.html`
- **功能**：用戶交互界面，支持員工名單管理、績效維度選擇及評語顯示。
- **關鍵區域**：
  - 员工姓名輸入（手動或自動匯入）
  - 評級選擇：特優/甲上/乙下等
  - 特質維度分類：思想、品德、能力等
  - 評語預覽區域

#### 前端邏輯：`static/js/performance.js`
- **核心功能**：
  - 管理員工選單與交互邏輯。
  - 處理維度特質選擇並傳送至後端API。
  - 控制評語生成流程（防止重複提交）。
- **主要方法**：
  ```javascript
  // 生成評語核心API呼叫
  async function generateComment() {
    const response = await fetch(apiurl_factory("comment/api/generate_comment/"), {...});
    // 處理回覆並更新UI顯示
  }

  // 批次評語匯出
  function downloadTxt/csv() { ... }
  ```

---

### 2.2 後端組件

#### API接口：`urls.py`
```python
# 評語生成API端點
path("api/generate_comment/", views.api_generate_comment, name="api_generate_comment")
```

#### 業務邏輯：`views.py`
- **主要功能**：處理評語生成請求、驗證輸入參數並呼叫LLM模型。
- **關鍵函數**：
  ```python
  @csrf_exempt
  @require_POST
  def api_generate_comment(request):
      # 輸入驗證與提示構建
      prompt = _build_prompt(...)
      llm = get_chat_model(model_type="LM_STUDIO")
      out = llm.invoke(prompt)
      return JsonResponse({"ok": True, "reply": reply})
  ```

#### 數據處理函數（`views.py`）
- `_norm_traits()`：清理重複特質並限制數量。
- `_build_prompt()`：構建軍事風格評語提示文字。

## 3. 整合流程

### 使用者體驗流程
1. 用戶選擇員工姓名並輸入（或匯入清單）
2. 選擇績效等級（如"甲上"）和維度特質（如"團隊合作"）
3. 前端提交JSON請求至後端API
4. 後端構建prompt並呼叫LLM生成評論
5. 系統顯示結果或自動批次匯出

### 技術整合圖

```
┌─────────────┐     ┌───────────────┐     ┌───────────────┐

└─────────────┘     └───────────┬───────┘     └───────────┘
                                │
                                ▼
                     ┌─────────────────┐
                     │ 績效評語數據庫 │ ← (未來擴展)
                     └─────────────────┘
```

## 4. 資料結構

### 前端狀態管理
```javascript
// performance.js中的主要狀態變量
const state = {
  currentStudent: "員工姓名",
  selectedTraits: new Set(["特質1", "特質2"]),
  performanceGrade: "甲上",
  comments: [{student: "...", comment: "..."}]

```

### API請求/回應格式
**請求（POST到`/comment/api/generate_comment/`）**：
```json
{
  "student_name": "王小明",
  "traits": ["團隊合作", "進取心強"],
  "performance_grade": "甲上",
  "max_chars": 80,
  "temperature": 0.7
}
```

**回應**：
```json
{
  "ok": true,
  "reply": "本期表現優異，團隊合作無間，建議繼續發揚...",
  "provider": "lm_studio"
}
```

## 5. 特色功能

### 5.1 特質維度管理
- 分類系統：思想、品德、能力等維度分類
- 正面/負面指標顯示（使用CSS類別`passitive`/`negitive`)
- 自訂特質支持（最多10個）

### 5.2 評語風格
- 軍事風格標準化評論，字數限制可調整（預設80字）
- 支持多種LLM提供商（LM_STUDIO/Ollama/OpenAI）

### 5.3 數據匯出
- 支援文本和CSV檔案格式
```javascript
function downloadCsv() {
  const csvContent = [csvHeader, header.join(","), ...rows].join("\n");

}
```

## 8. 操作規範

### 前端使用說明
1. 輸入員工姓名（手動或匯入檔案）
2. 選擇績效等級及維度特質（至少一個）
3. 點擊"生成評語"按鈕
4. 查看結果並可選擇匯出

### 後端API規範
- 請求方法：POST `/comment/api/generate_comment/`
- CSRF認證：必要時使用X-CSRFToken標頭
- 資料驗證：
  - 必須包含`student_name`
  - `traits`或`performance_grade`至少存在一個
